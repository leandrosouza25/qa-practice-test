# UI Testing Plan — Sauce Demo

## Plano de Testes

O objetivo deste plano é validar os principais fluxos da aplicação Sauce Demo, cobrindo autenticação, navegação, gerenciamento de produtos, carrinho e processo de checkout, considerando cenários positivos, negativos e testes exploratórios.

Os testes foram divididos em:
- Nível Obrigatório (fluxos principais)
- Nível Diferencial (responsividade, acessibilidade e automação)



## Cenários Principais

Durante a análise da aplicação, foram identificados os seguintes cenários críticos:

- Fluxo de autenticação (login/logout e sessão)
- Gerenciamento de produtos (visualização, ordenação)
- Gestão do carrinho (adição e remoção de itens)
- Processo completo de checkout
- Navegação entre páginas e controle de sessão
